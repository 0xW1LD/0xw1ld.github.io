## Manual Error Based SQLi
Looking around we can find the `Zone Minder` version is `v1.37.63`, checking for `CVEs` we can find [CVE-2024-51482](https://github.com/ZoneMinder/zoneminder/security/advisories/GHSA-qm8h-3xvf-m7j3) a `boolean based SQLinjection`. However we don't have to do this blind, instead we can do an `Error Based SQLi` because in the `log` tab we can actually see errors so we can do error based `SQLi` instead.

If we send the following request we get a `500` response.
