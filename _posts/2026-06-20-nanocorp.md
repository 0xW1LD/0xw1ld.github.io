# User
Since we know it's an `Active Directory` environment, let's attempt to upload a zip file containing malicious `ntlm theft` files. We can try the `ntlm theft` methods shown on [hacktricks](https://book.hacktricks.wiki/en/windows-hardening/ntlm/places-to-steal-ntlm-creds.html#ntlm_theft).
