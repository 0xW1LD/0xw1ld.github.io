## XOP/MTOM SSRF
In this case `Apache CXF` is the framework that is being used alongside the `Java API for XML Web Services(JAX-WS)` to design the frontend functionality of the application. Based on the [Apache CXF Security Disclosures](https://cxf.apache.org/security-advisories.html) We can find several vulnerabilities, having a quick read through them it seems that we might be able to exploit [CVE-2022-46364](https://cxf.apache.org/security-advisories.data/CVE-2022-46364.txt?version=1&modificationDate=1670944473000&api=v2) given that we're able to submit `XML` documents. However since the vulnerability specifies `MTOM` we'll have to use the `multipart/related MIME` type.
